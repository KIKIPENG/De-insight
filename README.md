# De-insight

一個跑在 terminal 裡的 AI 對話工具，給做視覺的人用的。

不是那種會說「好的，讓我幫你整理一下」的 AI。
它比較像一個會跟你吵架的朋友——你說了什麼模糊的東西，它會追問你到底是什麼意思。

想法永遠是你的，它只是幫你把那個還沒成形的東西說清楚。

---

## 能做什麼

**對話**——你說你的想法，它用你的語言跟你討論，不會突然冒出一堆理論術語。你問它意見，它會說，不會一直反問你。

**記住你說過的話**——對話過程中，它會把你的洞見、問題、反應記下來。下次聊天還記得你上次在想什麼。

**讀你的資料**——把 PDF 丟進去，它會建知識圖譜。討論的時候如果你的想法跟某個理論有關，它會自己去找。

**幫你寫論述**——跟它說「我想寫論述了」，它會從你們的對話裡整理命題，找理論支撐，幫你起草結構。每一步都等你確認才往下走。


## 裝起來

```bash
cd backend
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

然後設定 API key：

```bash
cp .env.example .env
# 打開 .env，填你的 key
```

支援 OpenAI、Anthropic、OpenRouter（DeepSeek）、MiniMax、Ollama、Codex CLI。
在 TUI 裡按 `ctrl+,` 可以切換。

---

## 跑起來

開兩個 terminal：

```bash
# 第一個：後端
cd backend && source .venv/bin/activate
uvicorn main:app --reload

# 第二個：TUI
source backend/.venv/bin/activate
python3 tui.py
```

---

## 用法

打字就好。想做特定的事，打斜線：

- `/save` — 手動存一條洞見
- `/memory` — 看和管理你的記憶
- `/import` — 丟 PDF 進知識庫
- `/search` — 搜知識庫
- `/mode` — 切感性/理性模式

快捷鍵：`ctrl+f` 匯入 PDF、`ctrl+k` 搜知識庫、`ctrl+m` 管理記憶。

---

## 技術

Textual TUI + FastAPI + LiteLLM。
記憶存在 SQLite，向量搜尋用 LanceDB，知識圖譜用 LightRAG。
